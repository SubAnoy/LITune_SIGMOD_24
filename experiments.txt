Static experiments: (for one dataset, 4 dataset in total: books, fb, OSM, MIX)
0. Pre-training RL:
python3 RL_controller_offline.py --RL_policy DDPG --load_model default --save_model True --Index ALEX --query_type balanced
(python3 RL_controller_offline.py --RL_policy DDPG --load_model default --save_model True --Index ALEX --query_type read-heavy)
(python3 RL_controller_offline.py --RL_policy DDPG --load_model default --save_model True --Index ALEX --query_type write-heavy)

1. Run the eval_single_budget, output to single_budget.txt (budget 50, 100, 150,200,250)
    sh eval_single_budget.sh balanced ALEX | tee ./logs/single_budget_balanced.txt &&
    sh eval_single_budget.sh read-heavy ALEX | tee ./logs/single_budget_read.txt &&
    sh eval_single_budget.sh write-heavy ALEX | tee ./logs/single_budget_write.txt
2. Run evals with infinite budget
    sh exe.sh balanced ALEX | tee ./logs/full_budget_balanced.txt &&
    sh exe.sh read-heavy ALEX | tee ./logs/full_budget_read.txt &&
    sh exe.sh write-heavy ALEX | tee ./logs/full_budget_write.txt
